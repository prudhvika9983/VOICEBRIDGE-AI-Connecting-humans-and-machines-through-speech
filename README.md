# VoiceBridge AI: Human-Machine Speech Interface

**VoiceBridge AI** is a real-time speech-to-text and text-to-speech interface designed to facilitate seamless communication between humans and artificial intelligence. Built with a **FastAPI** backend and a responsive **HTML5/CSS3** frontend, it leverages the **Google Gemini 1.5 Flash** model to provide intelligent, low-latency responses to voice commands.

## 🚀 Key Features
* **Real-Time Speech Recognition:** Utilizes the Web Speech API for high-accuracy Indian English (`en-IN`) voice capture.
* **Intelligent Conversational AI:** Integrated with Google Gemini for context-aware and concise machine responses.
* **Audio-Visual Feedback:** Provides instant text transcripts for both user input and machine output, followed by automated voice synthesis.
* **Optimized HMI:** Designed specifically as a Human-Machine Interface (HMI) for accessibility and industrial automation concepts.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Backend:** Python 3.10+, FastAPI, Uvicorn
* **AI Engine:** Google Generative AI (Gemini 1.5 Flash)
* **APIs:** Web Speech API (Recognition & Synthesis)

## 📋 Prerequisites
Before running the project, ensure you have:
1.  A valid **Google Gemini API Key**
2.    A valid **MURF AI API KEY**
3.  Python installed on your system.
4.  Chrome or a Chromium-based browser (for Web Speech API support).
